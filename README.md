# 🎬 Movie Recommendation System

## 📌 Overview
This project recommends movies based on similarity using NLP techniques.
link dataset--"https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata"

## ⚙️ Tech Stack
- Python
- Streamlit
- TF-IDF
- Cosine Similarity

## 🚀 Features
- Movie recommendation
- Poster display (TMDB API)
- Interactive UI

## 📂 Files
- app.py → Streamlit app
- notebook.ipynb → model building
- movies.pkl → processed data
- similarity.pkl → similarity matrix

## ▶️ How to Run
streamlit run app.py
